# Workout_diary_react
## Description
Project made for a Frontend Development class at the University. 
## Tech Stack
### Frontend:
- ReactJS
- Redux
- normalizr
- formik
- material-ui
- reselect
- dev-tools
- react router
### Backend:
- NodeJS
- express
- mongoose
### Database:
- MongoDB
